This is about internal and external CSS

Internal CSS

Internal CSS refers to defining CSS styles directly within the HTML file using the <style> tag. The CSS rules specified within the <style> tag apply only to the HTML document in which they are defined. This method allows you to have the CSS styles embedded within the HTML file itself. Internal CSS is typically placed within the <head> section of the HTML document, between the <style> opening and closing tags.


External CSS

External CSS involves defining CSS styles in a separate external stylesheet file, which is then linked to the HTML document using the <link> tag. The external CSS file contains all the CSS rules and styles that apply to multiple HTML documents. By using external CSS, you can maintain a centralized stylesheet that can be shared across multiple HTML files, making it easier to manage and update styles across your website.
